FAQ

Punya pertanyaan? Kemungkinan besar sudah ada jawabannya di sini.

Versi Minecraft apa yang didukung?
TeraHost mendukung semua versi Minecraft Java Edition mulai dari 1.8 hingga versi terbaru. Kami juga mendukung berbagai server software seperti Vanilla, Spigot, Paper, Purpur, Fabric, dan Forge.
Berapa lama proses aktivasi setelah pembayaran?
Server akan diaktifkan maksimal 30 menit setelah pembayaran dikonfirmasi. Biasanya proses ini lebih cepat jika admin sedang online. Kami beroperasi dari pagi hingga malam hari.
Apakah bisa upgrade paket di tengah periode?
Tentu bisa! Kamu bisa upgrade paket kapan saja dengan menghubungi admin. Biaya upgrade dihitung secara prorata sesuai sisa waktu berlangganan kamu.
Metode pembayaran apa yang tersedia?
Kami menerima pembayaran via transfer bank (BCA, BRI, Mandiri, BNI), GoPay, OVO, Dana, ShopeePay, dan QRIS. Hubungi admin untuk informasi rekening terbaru.
Apakah ada jaminan refund?
Kami menyediakan garansi refund dalam 24 jam pertama jika terjadi masalah teknis dari pihak kami yang tidak dapat diselesaikan. Hubungi admin untuk proses refund.
Apakah server bisa diakses dari luar negeri?
Ya, server TeraHost dapat diakses dari mana saja. Namun untuk pengalaman terbaik dengan ping terendah, kami merekomendasikan untuk pemain yang berlokasi di Indonesia.
